 Issue Events (Timeline)

/repos/{owner}/{repo}/issues/{issue_number}/events

Events include:

• opened
• closed
• reopened
• merged
• review_requested
• labeled
• unlabeled
• assigned

Each event includes:

• event

 Commit Data (PR Scope)

GET
/repos/{owner}/{repo}/pulls/{pull_number}/commits

Fields:

• sha
• commit.author.date
• commit.committer.date
• commit.message
• author.login



 Minimal Field Set for PR Activity Tracking

If your goal is PR analytics or DORA metrics, you only need:

• created_at
• merged_at
• closed_at
• state
• merged
• commits
• additions
• deletions
• reviews[].submitted_at
• events[].created_at
